# 爻算

一个有趣的命运解析H5应用，通过生辰推算你的爻像，并给出毒舌解读，还可以"逆天改命"！

## 功能特色

* 基于生日推算爻像，显示毒舌解读
* 炫酷的3D爻像展示
* "破命"功能，让你掌握自己的命运
* 生成分享海报，分享到朋友圈

## 技术栈

* 纯前端实现（HTML5/CSS3/JavaScript）
* Three.js 实现3D效果
* GSAP 实现动画效果
* Canvas 生成分享海报

## 使用方法

直接访问网站，输入生日，查看你的爻像和解读。点击"我命由我不由天，给我破！"体验逆天改命的快感！

## 注意事项

本应用纯属娱乐，解读内容不必当真 :)
